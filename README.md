# 🎬 BiblioCine App
Aplicación móvil para la gestión y exploración de catálogos de películas, construida con un enfoque en escalabilidad y código limpio.

A new Flutter project.

1. Copiar el env.template y renombarlo a .env
2. Cambiar las variables de entorno (The MovieDB)

## 🛠️ Stack Tecnológico

Este proyecto utiliza las herramientas más modernas del ecosistema Flutter para garantizar un rendimiento óptimo

* **Framework:** [Flutter](https://flutter.dev/)
* **Arquitectura:** Clean Architecture (Capas: Data, Domain, Presentation).
* **Gestión de Estado:** [Riverpod](https://riverpod.dev/) (Reactivo y seguro).
* **Persistencia Local:** [Drift](https://drift.simonbinder.eu/) (Motor SQLite potente).
* **Networking:** [Dio](https://pub.dev/packages/dio) (Cliente HTTP para consumo de APIs).
* **Navegación:** [GoRouter](https://pub.dev/packages/go_router) (Manejo de rutas declarativo).
